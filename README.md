# irbiddlecombe_com_blog
irbiddlecombe.com blog website


**setup**

- npx create-react-app react-irbiddlecombe-com
- cd react-irbiddlecombe-com/
- npm i bootstrap
- npm i react-router-dom
- nvm install 16 && nvm use 16
- npm start
